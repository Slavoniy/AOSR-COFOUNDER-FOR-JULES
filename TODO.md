## 📌 Правила для ИИ-агента (Jules)
1. Бери в работу строго одну задачу (один чекбокс `[ ]`) за раз.
2. После успешного тестирования и завершения задачи меняй `[ ]` на `[x]`.
3. Обязательно учитывай двойную логику базы данных (Postgres Pool + In-Memory массивы), как реализовано в `server.ts`.
4. Секреты (API ключи) бери только из `.env` или пример '.env.example'.
